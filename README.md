# Persyaratan Sistem
- flutter SDK, versi yang digunakan pada projek ini versi 3.10.3
- android studio
- Visual Studio Code

# instalasi
buka projek pada IDE yang digunakan,lakukan perintah 'flutter pub get' pada terminal

```sh
cd sea-sell-crypto-ui
flutter pub get
```
